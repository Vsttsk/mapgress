# Монитор Лента

Веб-приложение для менеджера Саши — мониторинг торговых комплексов «Лента» на карте.

## Структура проекта

```
map-lenta/
├── index.html
├── app.js
├── style.css
├── config.js
├── stores_final.csv
└── README.md
```

## Запуск

1. Откройте `index.html` в браузере (рекомендуется через локальный сервер).
2. Введите пароль: **Лента2024**
3. Используйте карту: кликайте на маркеры, отмечайте посещения.

### Запуск через локальный сервер (рекомендуется)

```bash
cd map-lenta
python3 -m http.server 8080
```

Откройте http://localhost:8080

## Функции

- **Карта** — 226 ТК с маркерами (зелёный = наш офис Максутов, синий = другие офисы, серый = нет офисов)
- **Панель ТК** — информация по клику на маркер
- **Отметка посещения** — форма с датой, присутствием офисов, комментарием
- **Статистика** — за 14 дней
- **Дашборд** — график посещений
- **LocalStorage** — сохранение данных локально

## Данные

`stores_final.csv` — формат:
- tk_number — номер ТК
- address — адрес
- lat, lng — координаты
- offices — список офисов (Максутов = наш офис)
